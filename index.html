kokoku.js

japanese: 広告, means advertisement

Simple JavaScript ad rotator

View project on GitHub Download kokoku.js

Privacy

kokoku.js doesn't require any foreign scripts. Don't let ad networks track data of your clients.

Size

kokoku.js is less than 1 KB minified and gzipped.

Browser support

kokoku.js works in every modern browser.

Multilingual ads

If you have a multilingual website, you can load banners in the local language of your visitors.

Free and open source

Source code released under terms of Creative Commons Zero 1.0.

Easy to use

All ads will be in a simple JSON file. You need only put div in place where you want to see ads.

How to use

  1. Load kokoku.js
    <script src="kokoku.js"></script>
  2. Create JSON file with your ads
    {
  "ad1": {
    "lang": "any",
    "code": "<a href='http://komachi.github.io/kokoku.js'><h1>kokoku.js is awesome!</h1></a>",
    "ping": "http://example.com/ad?id=ad1"
  },
  "ad2": {
    "lang": "en",
    "code": "<h1>fff</h1>"
  }
}
    lang is a language of the ad. Can be set to any. For language detection kokoku.js use document.documentElement.lang
    code is a code of the banner.
    ping is an URL which will requested after banner load.
  3. Load that json
    <link rel="kokoku" href="kokoku.json" type="application/json" />
  4. Put a div where you want to see banner
    <div data-kokoku="ad1,ad2"><div>
    data-kokoku contain a comma separated list of banners which you want use. One of that banners will be loaded. Can be set to any.
